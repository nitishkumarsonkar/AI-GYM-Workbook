# Product Context

Describes the user-facing value props derived from the project brief.

## Vision
- Provide a structured workout companion that syncs progress via Supabase.
- Ensure every user lands on authentication before accessing personalized dashboards.
- Enable users to receive recommendations tailored to their active routines.
- Maintain clear documentation of active goals and dependencies.
- Leverage Gemini LLM insights to recommend exercises aligned with user goals (e.g., fat loss, strength).

## Derived Requirements
1. **Auth-first navigation** as primary entry point.
2. **Workout plan personalization** per Supabase user enriched by their daily exercise logs.
3. **Knowledge continuity** through context files.
4. **Goal-aware AI recommendations** via Gemini to surface daily exercise suggestions.
